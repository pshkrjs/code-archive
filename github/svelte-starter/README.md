## Svelte Starter

Hello World app using [Svelte UI framework](https://svelte.technology/).

##### Instructions to run:
  
```
$ git clone https://github.com/pshkrjshnde/svelte-starter.git
$ cd svelte-starter
$ npm install
$ npm run build
```

Then, open index.html in a browser.
